# my-clash-rule

## 使用

### 直接引用 ini

https://raw.githubusercontent.com/kiritoxkiriko/my-clash-rule/refs/heads/main/my-clash-rule.ini

### Subconverter 示例

把上面的 ini 当作 config 参数传入：

http://127.0.0.1:25500/sub?target=clash&url=你的订阅链接（需URLEncode）&config=https://raw.githubusercontent.com/kiritoxkiriko/my-clash-rule/refs/heads/main/my-clash-rule.ini
